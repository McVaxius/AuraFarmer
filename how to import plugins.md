# How To Import Aura Farmer

## Dev Plugin Path

`Z:\AuraFarmer\AuraFarmer\bin\x64\Debug\AuraFarmer.dll`

## Add The Plugin In XIVLauncher

1. Launch FFXIV through XIVLauncher.
2. Open Dalamud settings with `/xlsettings`.
3. Go to `Experimental`.
4. Add `Z:\AuraFarmer\AuraFarmer\bin\x64\Debug\AuraFarmer.dll` to `Dev Plugin Locations`.
5. Open the plugin installer with `/xlplugins`.
6. Go to `Dev Tools > Installed Dev Plugins`.
7. Enable `Aura Farmer`.
8. Run `/aurafarmer` in game to open the main window.
9. Use `/aurafarmer config` if you want the settings window directly.
